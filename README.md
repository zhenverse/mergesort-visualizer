# Merge Sort Visualizer

A web-based interactive visualizer for the Merge Sort algorithm, built strictly with Vanilla JavaScript, HTML, and CSS. This project demonstrates the divide-and-conquer mechanism of Merge Sort through step-by-step DOM animations, highlighting the exquisite algorithmic logic.

This project is live at: [Merge Sort Visualizer](https://zhenverse.github.io/mergesort-visualizer).

## Features
- **Random Array Generation**: Generates a randomized set of bars (data points) with varying heights.
- **Algorithm Visualization**: Visually distinguishes the "compare", "overwrite", and "complete" phases of the Merge Sort process.
- **Modular Architecture**: Built with ES6 Modules, enforcing a unidirectional data flow from the sorting backend to the animation frontend.
